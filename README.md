# Dr. Fergus Hamilton — Academic Website

Static site for Dr. Fergus Hamilton’s academic profile, research highlights, publications, and contact details.

## Contents

- `index.html`: page content and structure
- `styles.css`: site styling
- `profile.jpg`: hero image

## Updating the site

1. Edit `index.html` to update copy, sections, or links.
2. Adjust `styles.css` for layout or visual changes.
3. Replace `profile.jpg` with an updated image if needed.

## Local preview

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
